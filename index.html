Welcome to Our Client Website!

This is an example of a client's website content. Imagine this is your customer's actual webpage.

The chat widget on the bottom right is embedded directly into this page. It has its tenant ID hardcoded, specific to this client.

Feel free to interact with the chat to see how messages are sent to your n8n webhook along with the tenant information.

How to use this example:

  1. Save this file: Save the entire content of this document as index.html.
  2. Open in browser: Open the index.html file in your web browser.
  3. Get an n8n Webhook URL: In your n8n instance, create a new workflow. Add a "Webhook" node (choose "Catch Webhook"). Copy the "Webhook URL" it provides.
  4. Configure the widget: In the <script> block below, replace "YOUR_N8N_WEBHOOK_URL_HERE" with your actual n8n Webhook URL. Also, set the TENANT_ID to something meaningful like "ClientA_Website".
  5. Expose User Information (Client's Website): For the chat widget to pick up user details, the client's website *must* define a global JavaScript object named window.n8nChatWidgetUser with the user's id, name, and email. This needs to be done *before* the chat widget script loads. 
    <script>
    // Example: Dynamically populate these values after a user logs in on the client's site
    window.n8nChatWidgetUser = {
        id: "user_12345",
        name: "John Doe",
        email: "john.doe@example.com"
    };
</script>
<!-- Then, your N8N Chat Widget embed code below -->
    If window.n8nChatWidgetUser is not set, or is missing properties, the chat will still function but without user-specific data in the payload.
  6. Test: Type a message in the chat widget and click "Send". Check your n8n workflow – you should see the incoming webhook request containing your message, the TENANT_ID, and now also the userId, userName, and userEmail if configured.
  7. To embed on a real site: Copy everything from <div id="n8n-chat-widget-root"></div> to the closing </script> tag and paste it just before the closing </body> tag of your client's HTML page. Remember to customize N8N_WEBHOOK_URL and TENANT_ID for each client! And importantly, ensure the window.n8nChatWidgetUser object is set by the client's site for logged-in users.
